# Fitness & Nutrition Tracker

Une app complète et local conçue pour le suivi quotidien de la nutrition, des entraînements et de l'évolution corporelle. Conçue avec React et optimisée pour une utilisation sur mobile.


## Installation & Lancement

1. Cloner le dépôt :
```bash
git clone [https://github.com/votre-nom/fitness-tracker.git](https://github.com/votre-nom/fitness-tracker.git)
Installer les dépendances :
```

2. Installer les dépandances :
```
bash
cd tracker-app
npm install
```

3. Lancer le serveur local de dev
```
bash
npm run dev
```

4. Compiler pour transformer en apk ou pour mettre en developpement :
```
bash
npm run build
```

## Info à savoir :

- en cas de mise à jour de l'application sauvegarder le profil afin de pourvoir l'importer dans la nouvelle version.
